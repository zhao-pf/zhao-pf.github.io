如何搭建服务器和抓取免流混淆参数

这篇教程涵盖了我近期研究的成果，抽着自己休息的时间来写一篇教程，如何搭建免流，自己抓取免流混淆参数，而这篇文章就是为了教会小白怎样拥有一个自己的免流服务器，下面开始教程

此教程为2018年2月27日编写–>2020年2月27日重新上传
安装锐速
首先我们安装锐速，直接复制代码到连接服务器回车
第一步:首先我们更换内核，粘贴回车即可，然后无须进行任何操作，等待即可直到出现以下界面
wget -N --no-check-certificate https://freed.ga/kernel/ruisu.sh && bash ruisu.sh

2.出现以上界面点击重新连接

3.接着我们输入锐速安装脚本，粘贴回车即可，然后无须进行任何操作，等待即可直到出现以下
wget -N --no-check-certificate https://github.com/91yun/serverspeeder/raw/master/serverspeeder.sh && bash serverspeeder.sh

4.接着运行锐速，粘贴回车即可，然后无须进行任何操作
service serverSpeeder start
安装以及配置SSR
安装SSR，用下面这个脚本安装，复制粘贴回车即可
yum -y install wget wget -N --no-check-certificate https://softs.fun/Bash/ssr.sh && chmod +x ssr.sh && bash ssr.sh
备用安装下载：
yum -y install wget
wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubi/doubi/master/ssr.sh && chmod +x ssr.sh && bash ssr.sh

10.之后代码就正式自动部署了，到下图所示的位置，提示你下载文件，输入:y

2.选择#text

3.选择这个

4.回到主页面后点击三角符号，开始抓包
三角符号变成正方形。 开始抓包！！！ 抓包的时候关掉其他软件，断掉wifi.打开你的手机卡免流的应用刷新一下页面走点流量后切回wi.cap软件

免流原理
我用自己的话解释一下吧
运营商为了给客户提供方便，提供了一些优惠政策，如：接收彩信、登陆掌厅免除流量费以及免收取流量费的其他业务。
