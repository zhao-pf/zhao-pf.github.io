WordPress博客的搭建

此教程主要包括：服务器的购买配置-WordPress搭建
教程6月30日写–>2020年2月28日修改上传

选择centos6x64
宝塔linux：点击进入官网
找到centos安装代码复制
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install.sh && sh install.sh


选择y继续安装
安装完成

之后我们把登录地址复制到浏览器中

输入账号以及密码

我们选择LNMP

接下来他会自己安装，我们可以点击右上角的x让他后台下载


关掉以后可以在这里看下载进度
接下来我们设置网站
https://cn.wordpress.org/txt-download/

大家可以进去下载最新版本

点击左边的文件之后我们上传即可注意上传到默认文件夹wwwroot内


上传完成之后点击解压


大功告成
接着让我们绑定网站

这里写你的域名，如果没有的话可以用你的服务器ip代替
我会用ip和域名分别演示
先是域名

我们点击数据库MySQL

添加数据库

域名的创建和ip创建一样


上面的图是我自己的域名，随便分配出来一个二级域名
我用的是阿里云，新用户第一年6元
www.Zhaopf.top

添加完成

输入你的服务器IP，回车GO

我们进入了博客配置

用户名即刚才写的wordpress
密码即刚才那个密码
如果你忘记可以按照下面方法查看


Ok

这些都是自行设置，我在这里也就不纠结设置什么了
直接安装


登录

到此，博客就已经创建成功了

输入服务器ip即可进入网站

博客的后台地址为http://45.77.211.249/wp-admin/
http://服务器IP/wp-admin/

我们可以写一篇文章


Ok，博客的搭建已经完成
Wordpress的其他功能什么的还是得自己探索
对了，建议最后把宝塔的密码以及网页端口修改
